weblog
======

个人博客。

[Example: 马孔多的主页](http://whoop.vipsinaapp.com/blog/)

Download
--------
    $ git clone https://github.com/ghostrong/weblog.git

Quick Start
-----------

- 配置数据库连接参数：config.py
- 创建table

    $ python data.py
- 运行 app

    $ python myapp.py

通过Admin 管理数据
------------------

    http://localhost:5555/blog/admin/

注意根据需要修改访问权限。
